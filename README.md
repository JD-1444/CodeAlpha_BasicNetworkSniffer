## CodeAlpha_BasicNetworkSniffer
# Project Overview

This project was developed as part of the CodeAlpha Cyber Security Internship. The objective of this project is to capture and analyze network packets using Python and Scapy.

# Objectives
- Capture network traffic packets.
- Analyze packet structure and content.
- Understand network communication.
- Display source and destination IP addresses.
- Identify network protocols.

# Technologies Used
- Python
- Scapy
- Npcap
Visual Studio Code

# Features
- Captures live network packets.
- Displays source IP address.
- Displays destination IP address.
- Identifies protocol type (TCP, UDP, ICMP).
- Displays packet payload information.
- Helps understand network traffic flow.

# Installation
Install Scapy:

pip install scapy

# Install Npcap:

Download and install Npcap from https://npcap.com

Usage

Run the program:

python sniffer.py

Generate network traffic by opening websites such as Google or YouTube.
